
# Portfolio

This is my personal portfolio - build with Next.js project that will be hosted in an S3 bucket and will utilize Sanity.io as a CMS.

## Tech Stack
- React
- Next.js
- Typescript
- Tailwindcss
- GraphQL (Sanity.io)
- AWS CLI
- AWS S3
- AWS Route53
- Terraform

## Run Locally

Clone the project

Install dependencies

```bash
  yarn
```

Run tests

```bash
  yarn test
```
